# Meu Projeto de Análise de Dados
Este repositório contém um conjunto de dados simples chamado gasolina.csv, que registra o preço da gasolina ao longo de 10 dias consecutivos.

O objetivo é demonstrar uma análise básica utilizando Python, Pandas, Seaborn e Matplotlib para:

- Ler os dados da série temporal

- Gerar um gráfico de linha mostrando a variação do preço diário

- Salvar o gráfico como gasolina.png

- Organizar o código em um script chamado gasolina.py
